# Variation 1: iPad版デモアプリ

## 概要

iPadの実際のマルチタッチ入力を使用したVariation 1のデモアプリです。

## 本来のVariation 1の動作

1. **4本指を同時にタッチ** → システムが4本の指を認識・識別
2. **特定の指を離す** → 残った指の組み合わせでモードが決まる
3. **残った指で操作** → そのモードで操作を実行

例：
- 4本指タッチ → 人差し指以外を離す → 人差し指だけで操作（ポインティングモード）
- 4本指タッチ → 薬指と小指を離す → 人差し指+中指で操作（スクロールモード）

## 使い方

### iPadでの使用

1. iPadのSafariでこのページを開く
2. 画面に**4本指を同時にタッチ**（人差し指、中指、薬指、小指）
3. **特定の指を離す**
4. ジェスチャが自動検出される

### ローカルサーバーで起動

```bash
cd ipad-demo
python -m http.server 8000
```

iPadのSafariで `http://[PCのIPアドレス]:8000` を開く

## 対応ジェスチャ

### V1-1: 単一指モード（4本→1本）
- **V1-1a**: 人差し指のみ → 通常ポインティング/選択モード
- **V1-1b**: 中指のみ → スクロールモード
- **V1-1c**: 薬指のみ → 描画/ペンモード
- **V1-1d**: 小指のみ → 消しゴム/削除モード

### V1-2: 2本指モード（4本→2本）
- **V1-2a**: 人差し指+中指 → 2本指スクロール
- **V1-2b**: 人差し指+薬指 → ピンチズーム
- **V1-2c**: 人差し指+小指 → 回転操作
- **V1-2d**: 中指+薬指 → パン操作
- **V1-2e**: 中指+小指 → ブラシサイズ調整
- **V1-2f**: 薬指+小指 → 不透明度調整

### V1-3: 3本指モード（4本→3本）
- **V1-3a**: 人差し指+中指+薬指 → 3本指スワイプ（ウィンドウ切替）
- **V1-3b**: 人差し指+中指+小指 → 3本指ドラッグ
- **V1-3c**: 人差し指+薬指+小指 → 3本指タップ（特殊機能）
- **V1-3d**: 中指+薬指+小指 → 3本指ピンチ

## 指識別の仕組み

1. **4本指タッチ検出**: 4つのタッチポイントを検出
2. **x座標でソート**: 左から右へ並べる
3. **指を割り当て**:
   - 一番左 → 人差し指
   - 左から2番目 → 中指
   - 左から3番目 → 薬指
   - 一番右 → 小指
4. **残った指を追跡**: 初期位置から移動を追跡
5. **ジェスチャ判定**: 残った指の組み合わせから判定

## 注意事項

- iPadの設定で「マルチタスク用ジェスチャ」をオフにすることを推奨
- 4本指を**横一列に並べて**タッチすると認識精度が上がります
- 指を離すときは、残したい指の位置を動かさないようにしてください

## Study 1での使用

このアプリを使って、被験者に実際のジェスチャを試してもらい、評価を収集できます。
